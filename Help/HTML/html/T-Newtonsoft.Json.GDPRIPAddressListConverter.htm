GDPRIPAddressListConverter Class

Provides a JSON converter for lists of IP addresses (IPv4 and IPv6), ensuring GDPR-compliant encryption and decryption of data with a rotating IV for enhanced security. This converter encrypts lists of IP addresses during serialization and decrypts them during deserialization, making it suitable for handling sensitive network configuration data or user IP logs.

Namespace:  Newtonsoft.Json
Assembly:  Walter.Cypher.Newtonsoft.Json (in Walter.Cypher.Newtonsoft.Json.dll)

Syntax

C#
[FeatureTagsAttribute]
public class GDPRIPAddressListConverter : BasicJsonConverter<List<IPAddress>, GDPRIPAddressListConverter>

Remarks

The GDPRIPAddressListConverter secures lists of IP address data by encrypting the values when serialized to JSON and decrypting them back into their original form upon deserialization. It supports both IPv4 and IPv6 address formats. Utilizing a specified password along with a rotating initialization vector (IV), which is stored alongside the cypher bytes, ensures that identical lists of IP addresses result in different encrypted outputs each time, significantly enhancing data security.

///

Rotating IVs make the encryption scheme more robust by preventing various cryptographic attacks and making the encrypted data less predictable. The management of the IV is fully automated and included in the encrypted data, ensuring seamless encryption and decryption processes without additional setup.

///

This converter is especially recommended for applications that need to process or store lists of IP addresses, which could be considered personally identifiable information (PII) under GDPR or similar privacy regulations. It aids in maintaining user privacy and ensuring compliance with legal requirements regarding data protection.

Examples

This example shows how to use the GDPRIPAddressListConverter on a property representing a list of IP addresses for JSON serialization.
C#
public class NetworkAccessList
{
[JsonConverter(typeof(GDPRIPAddressListConverter))]
public List<IPAddress> AllowedIPs { get; set; }
}
///
var accessList = new NetworkAccessList
{
AllowedIPs = new List<IPAddress> { IPAddress.Parse("192.168.1.1"), IPAddress.Parse("10.0.0.1") }
};
///
var options = new JsonSerializerOptions
{
Converters = { new GDPRIPAddressListConverter() }
};
var json = JsonSerializer.Serialize(accessList, options);
// The 'AllowedIPs' list is encrypted with a unique IV for each IP address, resulting in different cypher text for identical IP addresses.
