GDPRIPAddressConverter
Provides a JSON converter for IP addresses (IPv4 and IPv6), ensuring GDPR-compliant encryption and decryption of data with a rotating IV for enhanced security. This converter encrypts IP addresses during serialization and decrypts them during deserialization, making it suitable for handling sensitive information related to network configurations or user IP data.
GDPRIPAddressListConverter
Provides a JSON converter for lists of IP addresses (IPv4 and IPv6), ensuring GDPR-compliant encryption and decryption of data with a rotating IV for enhanced security. This converter encrypts lists of IP addresses during serialization and decrypts them during deserialization, making it suitable for handling sensitive network configuration data or user IP logs.
GDPRObfuscatedDateTimeConverter
Provides a JSON converter for [DateTime], ensuring GDPR-compliant obfuscation and de-obfuscation of date and time data with enhanced security. This converter applies cryptographic obfuscation to [DateTime] values during serialization and reverses the process during deserialization, making it ideal for protecting sensitive date-related information such as birthdates or event dates.
GDPRObfuscatedIntConverter
Provides a JSON converter for integers, ensuring GDPR-compliant obfuscation and de-obfuscation of numeric data with enhanced security. This converter applies cryptographic obfuscation to integer values during serialization and reverses the process during deserialization, making it ideal for protecting sensitive numeric information such as user IDs, order numbers, or quantities that could be linked to personal data.
GDPRObfuscatedStringConverter
Provides a JSON converter for strings, ensuring GDPR-compliant obfuscation and deobfuscation of textual data with enhanced security. This converter applies cryptographic obfuscation to string values during serialization and reverses the process during deserialization, making it ideal for protecting sensitive information such as names, email addresses, or any other personal identifiers.
