GDPRObfuscatedDateTimeConverter Class

Provides a JSON converter for [DateTime], ensuring GDPR-compliant obfuscation and de-obfuscation of date and time data with enhanced security. This converter applies cryptographic obfuscation to [DateTime] values during serialization and reverses the process during deserialization, making it ideal for protecting sensitive date-related information such as birthdates or event dates.

Namespace:  Newtonsoft.Json
Assembly:  Walter.Cypher.Newtonsoft.Json (in Walter.Cypher.Newtonsoft.Json.dll)

Syntax

C#
[FeatureTagsAttribute]
public sealed class GDPRObfuscatedDateTimeConverter : BasicJsonConverter<Nullable<DateTime>, GDPRObfuscatedDateTimeConverter>

Remarks

The GDPRObfuscatedDateTimeConverter enhances the privacy and security of date and time information by applying a cryptographic obfuscation technique when serialized to JSON, and de-obfuscating upon deserialization. It ensures that sensitive temporal data, which could be directly associated with an individual's personal data, is not exposed in plain text.

Utilizing a rotating initialization vector (IV) for each operation, the converter guarantees that the same [DateTime] value will result in different encrypted outputs, thereby significantly enhancing the security and unpredictability of the data.

This converter is particularly useful in scenarios where date and time information needs to be transmitted or stored in a format that complies with GDPR or similar data protection regulations, ensuring that personal data is handled with the highest level of security.

Examples

This example demonstrates how to use the GDPRObfuscatedDateTimeConverter for a property representing a sensitive date in a user profile for JSON serialization.
C#
public class UserProfile
{
    [JsonConverter(typeof(GDPRObfuscatedDateTimeConverter))]
    public DateTime DateOfBirth { get; set; }
}

var userProfile = new UserProfile
{
    DateOfBirth = new DateTime(1990, 1, 1)
};

var options = new JsonSerializerOptions
{
    Converters = { new GDPRObfuscatedDateTimeConverter() }
};
var json = JsonSerializer.Serialize(userProfile, options);
// The 'DateOfBirth' is obfuscated, enhancing the privacy and security of the serialized data.
